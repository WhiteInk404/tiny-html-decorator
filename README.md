# tiny-html-decorator
Using for improving Wechat public platform articles' layout   
It can decorate the plain html (no body and html tag) generated by MWeb  
Make it for [tesths](https://github.com/tesths)  
  
用来给公众号排版的小玩意儿  
给直接由markdown输出的html页面加点装饰  
给巨总帮忙写的  

## tutorial
**Without using command line**  
**Without editing module html file**  

**不需要用命令行执行python脚本**   
**也不需要将文章拷贝到写好的模板的指定位置上**  

FIRST: Save the outer html text generated by MWeb to a html file
SECOND: Drag that file into [this page]().  
THIRD: Ctrl+A, Ctrl+C, Ctrl+V to your Wechat background editor :)  

只需要  
1. 把输出的html存到文件  
2. 把文件拖到页面里  
3. 全选复制，粘贴到文章编辑器里  

## about
You can also download this page and edit style by yourself  
No other resources loaded for this page except a cdn zepto.js file  
I draw background pattern by css :)  

当然你可以把页面下载到本地，自己调整CSS  
除了CDN上的zepto.js，没有用其他资源了，背景是用CSS画的
