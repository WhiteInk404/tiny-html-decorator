# 简陋的排版工具  
用来做公众号文章排版的小玩意儿  
改善下直接由markdown转换成html页面的朴素排版  
肥肠简单，没有选择功能只有固定的样式，还凑会儿吧  
给[巨总](https://github.com/tesths)帮忙写的  
起因是使用markdown here生成的默认样式已不能满足这届粉丝的审美  

## 说明书  
**不需要用命令行执行python脚本**  
**不需要在html模板上拷贝来拷贝去**  
  
把大象装进冰箱需要几步? 答:3步  
1. 把markdown编辑器导出的html存到文件  
2. 把文件拖到[页面](https://nondanee.github.io/tiny-html-decorator/)里  
3. 全选复制，粘贴到公众号文章编辑器里  
  
## 关于  
多亏H5中新的文件操作特性，读文件也不需要后台，这样所以可以直接挂在Github Page上  
[nondanee.github.io/tiny-html-decorator/](https://nondanee.github.io/tiny-html-decorator/)  
如果爱折腾，把页面下载到本地，自己调整CSS  
除了CDN上的zepto.js，没有用链接其他资源，为了瘦身，背景是CSS画的:)  
至于为什么用zepto，假装自己不用jQuery，次爪爪
